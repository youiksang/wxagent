# 노트북LM → PPTX 변환기

구글 노트북LM으로 작성한 이미지 슬라이드 PDF를 편집 가능한 PowerPoint(PPTX)로 변환하는 앱입니다.

## 기능

- **PDF 불러오기**: 변환할 PDF 파일 선택
- **저장 경로 지정**: PPTX 파일 저장 위치 및 파일명 지정
- **폰트 선택**: 나눔고딕 또는 맑은 고딕 선택
- **OCR 변환**: EasyOCR로 한글/영어 텍스트 추출 후 편집 가능한 텍스트로 삽입

## 설치

```bash
pip install -r requirements.txt
```

> **참고**: numpy는 OpenCV 호환을 위해 2.x 미만 버전이 필요합니다. `numpy<2`가 requirements에 포함되어 있습니다.

## 실행

```bash
python notebooklm_to_pptx.py
```

## 사용 방법

1. **PDF 불러오기** 버튼으로 변환할 PDF 파일 선택
2. **경로 지정** 버튼으로 저장할 PPTX 파일 경로 지정 (자동 제안됨)
3. **폰트** 드롭다운에서 나눔고딕 또는 맑은 고딕 선택
4. **변환** 버튼 클릭
5. 슬라이드별 OCR 처리 후 PPTX 저장 완료

## 기술 사양

- **슬라이드 비율**: 16:9 (10 x 5.625 inches)
- **좌표 변환**: 이미지 픽셀 → PPT 인치 비율 기반 변환
- **텍스트 처리**: 기존 이미지 텍스트를 배경색으로 덮은 후, 추출된 텍스트를 원래 위치에 삽입
